Progressive Overload Rules & Guidelines

1. Double Progression Model

Each exercise lists a target **Rep Range** (e.g., 8-12 reps). Choose a starting weight. If you can perform the top of the range (12 reps) for **all** sets, the application will automatically increase the weight by 2.5 kg (or 5 kg for compound barbell exercises) next week, and drop your target reps to the bottom of the range (8). Work your way up in reps before adding weight again.

2. Injury Prevention & Safety Deloads

Because you are carrying 112.8 kg, your joints and connective tissue undergo high mechanical stress. If your reps drop below the minimum limit (e.g. less than 8 reps) on any set, the system automatically triggers a **10% deload** for the next session. This resets muscle fatigue, ensures flawless execution form, and allows joint recovery.

Google Sheets Sync Integration

Keep your training logs synchronized to a personal Google Sheet automatically. Setting it up takes less than 2 minutes:

1. Set Up Apps Script

  1. Create a new Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Clear any template code, paste the script on the right, and click Save.
  4. Click Deploy > New Deployment. Select Web App.
  5. Under access, select "Anyone", click deploy, and authorize permissions.
  6. Copy the generated Web App URL.

2. Paste Gateway URL

Apps Script Code:

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(["Date", "Workout Name", "Exercise Name", "Set Number", "Weight (kg)", "Reps"]);
    }
    data.exercises.forEach(function(ex) {
      ex.sets.forEach(function(set, idx) {
        if (set.reps !== null && set.reps !== undefined) {
          sheet.appendRow([data.date, data.workoutName, ex.name, idx + 1, set.weight, set.reps]);
        }
      });
    });
    return ContentService.createTextOutput(JSON.stringify({ status: "success" })).setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: error.toString() })).setMimeType(ContentService.MimeType.JSON);
  }
}
function doGet() {
  return ContentService.createTextOutput("Aesthetix Sync Gateway is Active!");
}
